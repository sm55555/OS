### tar로 압축

fonts 폴더를 fonts.tar로 폴더로 압축하기

```
tar -cvf fonts.tar fonts 
```

### tar 압축 해체

fonts.tar 압축 풀어서 fonts 폴더로 만들기

```
tar -xvf fonts.tar
```


### tar.gz 압축

fonts 폴더를 fonts.tar.gz로 압축하기

```
tar -cvfz fonts.tar.gz fonts
```

### tar.gz 압축 해체

fonts.tar.gz 압축 풀어서 fonts 폴더로 만들기

```
tar -xvfz fonts.tar.gz
```
