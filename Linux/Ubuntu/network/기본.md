## Ubuntu20.04

vi /etc/netplan/50-cloud-init.yaml

```
network:
  ethernets:
    ens160:
      addresses:
      - 192.168.1.60/24
      gateway4: 192.168.1.1
      nameservers:
        addresses:
        - 8.8.8.8
        search:
        - 8.8.4.4
  version: 2
```

### Netowrk 활성화 

ip a 네트워크 확인 후

```
ip link set eno1 up
```

### bonding 해체

vi /etc/netplan/50-cloud-init.yaml 

bonding 관련 주석 후 일므에ㅏ

```
ip link delete dev bond0
```
